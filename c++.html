C++ is a general-purpose object orientated programming language, that later in 1998 became an ISO standard. It was developed by Bjarne Stroustrup in the early 1980s while he was working at AT&T Bell Labs, and was orignally named 'C with classes'.

C++ was created after adding extra features to the C programming language in order to make it an efficient object-oriented programming language (OOP). His language included some add on features such as: classes, basic inheritance, default function arguments, and polymorphism. Stroustrup has stated that the purpose of the C++ language is to make writing good programs easier and more pleasant for the individual programmer.

There are many different versions of the C++ programming language:

  • C++98
  • C++11
  • C++14
  • C++17 - The latest edition released in 2017
