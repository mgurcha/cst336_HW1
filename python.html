Python was created in December 1989 by Guido van Rossum, the Python community's Benevolent Dictator for Life. It started out as a hobby project to work in during Christmas vacation, and was born out of the ABC language. The ABC language was a terminated project of the Dutch CWI research institute, whic is where van Rossum worked.

Python is also a interpreted language, which means that it does not need to be compiled in order to be executed. It is high level language, which allows a programmer to focus on what to do instead of how to do it. Python is also a excellent programming language for beginners due to its simple syntax when being compared to other languages. It hs many great feaures, such as:

  • Simple to learn and easy to read
  • Extensive Support Libraries
  • Open Source - meaning it is free to use and distribute (even for commercial purposes)
  • User-friendly Data Structurese
