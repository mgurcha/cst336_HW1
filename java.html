Java is another object-oriented programming language. It was developed in the early 1990s by a small team of engineers working at Sun Microsystems at the time. THe team consisted of James Gosling (the main developer), Mike Sheridan, and Patrick Naughton. The team was known to others as the 'Green Team'. It was initially called the Oak Programming Language.

The language borrows much syntax from both C and C++, but has a simpler object model. Java is also a High Level Programming Lanuage. It was developed according to the principle of WORA (Write Once Run Anywhere). It was introduced to the public in 1995 and is now owned by Oracle. The 'Green Team' had five goals in mind when designing the language:

  • To keep it simple to use, familiar to C++/C programmers, and be object oriented
  • Keep it both reliable and secure from outsiders
  • Keep it 'architecture-neutral' and portable, which is possible through JVM (Java Vitrual Machine)
  • Executable with High Performance - it includes features such as 'automatic garbage collection'
  • Interpreted, threaded and dynamic - the Java compiler can execute Java 'bytecodes' on any machine where the interpreter and run time system is installe
